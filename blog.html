Important FAQs

What is the Document Object Model (DOM)?

DOM is essential for creating dynamic and interactive web pages. It provides a way for JavaScript to communicate with the HTML structure and manipulate it to achieve desired effects.

How do you select an element from the DOM?

To select an element from the DOM, you can use several methods provided by JavaScript- getElementById(), getElementsByTagName(),getElementsByClassName(), querySelector(), and querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to handle events on its child elements. This improves performance, handles dynamically added elements, and simplifies code by reducing the number of event listeners.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles-setAttribute() and removeAttribute() to modify attributes like href, disabled, etc. To change inline styles, directly access the style property. For class-based styling, use classList.add(), classList.remove(), and classList.toggle(). By combining these methods, JS can dynamically alter the appearance and behavior of web elements.
